Proporciones con fracciones (fr)

La unidad fr reparte el espacio libre de forma proporcional. 1fr 1fr 1fr reparte 3 partes iguales.

Todas las columnas con fr
Ejercicio: cambia a 2fr 1fr 1fr y ve cómo se redistribuye el espacio.
1fr
1fr
1fr
.fr-grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* prueba 2fr 1fr 1fr */
  gap: 0.75rem;
}
Combinando % y fr
Ejercicio: cambia el 30% por otro valor y observa cómo cambia el resto de las columnas.

2fr
