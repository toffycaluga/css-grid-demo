Layout completo con grid-template-areas

Nombramos cada sección del layout y luego “dibujamos” la estructura con grid-template-areas.

Layout: Header, Sidebar, Main, Footer
Ejercicio: agrega una nueva área (por ejemplo “aside2”) y cambia el diseño solo modificando grid-template-areas.
Header
Menú principal del sitio
Sidebar
Enlaces, filtros, menú secundario
Main
Contenido principal de la página. Aquí iría el texto, las tarjetas, etc. Piensa en la sección de “contenido” de un sitio real.
Footer
Información de contacto, links legales, redes sociales, etc. 
.layout-grid {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  grid-template-columns: 220px 1fr;
  grid-template-rows: auto 1fr auto;
  gap: 0.75rem;
  min-height: 60vh;
}

.layout-header  { grid-area: header; }
.layout-sidebar { grid-area: sidebar; }
.layout-main    { grid-area: main; }
.layout-footer  { grid-area: footer; }
