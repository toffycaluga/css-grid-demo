Galería de imágenes con Grid

Usamos Grid para crear una galería simple. Las imágenes se adaptan al ancho de la columna.

Galería
Ejercicio: cambia el número de columnas y el tamaño de las imágenes. Juega con object-fit. 
.gallery-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* cambia a 2, 4, etc. */
  gap: 8px;
}

.gallery-grid img {
  width: 100%;
  height: 200px;
  object-fit: cover; /* cover, contain, fill */
}
