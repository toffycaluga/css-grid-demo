¿Cuándo usar Flexbox y cuándo Grid?

Usa Flexbox para layouts en una sola dirección (fila o columna) y Grid cuando necesites trabajar con filas y columnas al mismo tiempo.

Ejemplo Flexbox: barra de navegación
Layout unidimensional: solo una fila con elementos alineados.
